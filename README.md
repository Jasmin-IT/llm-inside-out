# LLM Inside Out

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/d0aa35e1-e32a-456f-aeb0-ce58405e6b12

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Deploy

This is a static Vite build. To publish it anywhere:

1. Build: `npm run build`
2. Upload the `dist/` folder to your host (Vercel/Netlify/Cloudflare Pages/S3, etc.)
